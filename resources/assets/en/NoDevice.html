Please check the followings when device is not found

Device

  • Make sure the device is powered on
  • Make sure the device is connected to network
  • You would need to turn on "Network Standby" function in order to turn power on the device from App.

Smartphone

  • Make sure the device and the phone is connected to same network

Router

  • Make sure the router is powered on
  • Please try plugging/unplugging the power supply of wireless LAN router and the device, check the wireless LAN router power-on status, and try reboot of the wireless LAN router
